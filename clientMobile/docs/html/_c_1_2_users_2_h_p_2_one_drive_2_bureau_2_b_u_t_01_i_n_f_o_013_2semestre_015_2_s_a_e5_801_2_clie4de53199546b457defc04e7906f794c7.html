Composant BilletDetails.

Composant BilletDetails.Ce composant affiche les détails d'un billet de réservation, y compris les informations sur le voyage, les bagages et l'accompagnateur. Il permet également de supprimer la réservation via un appel API.

@component

const billet = { num_reservation: "101", lieu_depart: "Paris", lieu_arrivee: "Lyon", heure_depart: "2025-01-26T08:00:00Z", heure_arrivee: "2025-01-26T12:00:00Z", wheelchair: "RM", additionalInfo: "Besoin d'assistance spéciale.", numBags: 2, hasCompanion: true, companion: { name: "John", surname: "Doe", phone: "1234567890", email: "john.doe@example.com" }, bagages: [ { weight: "12", description: "Valise rouge" }, { weight: "8", description: "Sac à dos noir" }, ], }; return <BilletDetails billet={billet} >;

Paramètres
{Object}props - Propriétés du composant.
{Object}props.route - L'objet de navigation contenant les paramètres de la route.
{Billet}props.route.params.billet - Les détails du billet.
Renvoie
{JSX.Element} Le composant BilletDetails.
