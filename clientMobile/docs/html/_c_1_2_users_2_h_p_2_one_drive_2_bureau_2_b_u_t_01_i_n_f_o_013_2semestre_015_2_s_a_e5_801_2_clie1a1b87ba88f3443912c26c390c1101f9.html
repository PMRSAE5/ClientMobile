Composant Connexion.

@component

return ( <Connexion navigation={navigation} onLoginSuccess={onLoginSuccess} > )

Paramètres
{Object}props - Les propriétés du composant.
{Object}props.navigation - L'objet de navigation pour naviguer entre les écrans.
{Function}[props.onLoginSuccess] - Fonction appelée après une connexion réussie.
Renvoie
{JSX.Element} Le composant de connexion.
