Composant principal Reservation2.\+Permet à l\textquotesingle{}utilisateur de fournir des informations supplémentaires pour personnaliser son assistance lors de son voyage, telles que \+:
\begin{DoxyItemize}
\item Les détails d\textquotesingle{}un accompagnateur (nom, prénom, téléphone, email).
\item Le nombre de bagages.
\item Les options liées à l\textquotesingle{}utilisation de fauteuils roulants.
\item Une description additionnelle pour des besoins spécifiques. Les informations collectées sont ajoutées au billet existant et transmises à l\textquotesingle{}étape suivante.
\end{DoxyItemize}

@description Fonctionnalités principales \+:
\begin{DoxyItemize}
\item Collecte des informations sur un éventuel accompagnateur.
\item Spécification du nombre de bagages transportés.
\item Sélection d\textquotesingle{}une option liée à l\textquotesingle{}utilisation de fauteuils roulants.
\item Fourniture d\textquotesingle{}informations complémentaires dans un champ texte.
\item Transition avec un écran de chargement initial simulant un temps de traitement.
\item Validation des entrées utilisateur (exemple \+: vérifier que le nombre de bagages est spécifié).
\item Navigation vers la prochaine étape avec les données mises à jour.
